Arducom - Arduino Serial Communication Library

Arducom - Serial Communication Library

How To Use?

At First Download the files and copy the C++ library to your working directory.Here Arducom is the name of class and Ser is name of Object. COM4 is the name of Serial Port. Incase of linux We Use dev/ttyACM0 or something similar. First see in which serial port arduino is communicating with your computer.

Code:

#include"src/Arducom.h"
using namspace ac;
Arducom Ser("/dev/ttyACM0");

getPort()

This function simply returns you some string. The string will be the port your are Currently using.This will print Current Port you are working with.

Code:

std::string CurrentPort = Ser.getPort();
cout << CurrentPort << endl;

printPort()

This fuction will print the port you are working with on the screen.

Code:

Ser.printPort();

resetPort()

This funciton resetPort is used to reset the port to another name. It takes two arguments. Where first argument is name of port and the second one in integer. 0 or 1. 1 denotes the verbose on and 0 for verbose off. Simply 0 will not print anything and 1 will print Some informations.

Code:

Ser.resetPort("/dev/ttyACM0",0);
or Ser.resetPort("/dev/ttyACM0",1);

sendBuffer()

This function is used to send information over seial port. It takes one parameter. You can send both integer and string variable to this function. It returns 0 or 1 with success or failure of sending the information.

Code:

Ser.sendBuffer("Hello World");
Ser.sendBuffer(1);
